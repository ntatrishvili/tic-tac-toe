# Tic-Tac-Toe (React)

This is a simple Tic-Tac-Toe game built with React as a small practice project.  
It lets two players take turns placing X and O on a 3Ã—3 grid until there is a winner or a draw.

## Running the project

1. Install dependencies:
```
npm install
```

2. Start the development server:

```
npm start
```
